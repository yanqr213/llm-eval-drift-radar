# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-08

### Added

- Initial standard-library Python CLI.
- JSONL and CSV input support.
- Case alignment by `case_id`.
- Regression and drift detection for failures, fixes, scores, categories, models, latency, and cost.
- Markdown, JSON, and CSV report outputs.
- Thresholds JSON and CI `--check` mode.
- Examples, documentation, tests, and GitHub Actions workflow.
